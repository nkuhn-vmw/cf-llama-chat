Channels

What it does

Channels are real-time group messaging spaces where you can collaborate with other users. Messages stream live via SSE so everyone in a channel sees updates instantly.

How to use it

  • Enter a name and optional description, then click Create to start a new channel.
  • Click any channel in the list to open it and view messages.
  • Type a message and press Enter or click Send to post.
  • Messages appear in real-time for all participants.
Tip: Only the channel creator or an admin can delete a channel. Deleting a channel removes all its messages permanently.

Notes

What it does

Notes let you create and organize personal text entries with Markdown support. They're scoped to your account so only you can see them.

How to use it

  • Click New Note to start writing.
  • Give your note a title and write content in the editor. Markdown syntax is supported.
  • Click Show Preview to see how your Markdown will render.
  • Click Save Note to persist your changes.
  • Select an existing note from the sidebar to edit or delete it.
Tip: Notes are personal and private to your account. Use them to save research, meeting notes, or reference material you want to keep handy.

Memory

What it does

Memory entries are pieces of information the AI remembers about you across conversations. They help the AI provide more personalized and context-aware responses.

Categories

  • Fact — Things that are true about you or your work (e.g., "I use PostgreSQL for my database").
  • Preference — Your personal preferences (e.g., "I prefer concise answers").
  • Instruction — Standing instructions for the AI (e.g., "Always include code examples").
  • General — Anything else worth remembering.

How to use it

  • Write a memory entry, select a category, and click Add Memory.
  • Use the search bar to find specific memories by keyword.
  • Use the filter buttons to narrow by category.
  • Click Delete on any memory to remove it.
Tip: The AI automatically includes your memories as context in conversations. Be specific with your entries for the best results.

Prompts

What it does

Prompt presets let you save reusable prompt templates. They support template variables that get filled in at use time, and can be shared with your team.

Template variables

Use {{variable_name}} syntax in your prompt content. When you use the prompt, you'll be asked to fill in each variable. For example:

  • Summarize the following text in {{language}}: {{text}}
  • Write a {{tone}} email about {{topic}}

How to use it

  • Fill in the title, optional command (e.g., /summarize), and prompt content.
  • Check Shared with team to make the prompt available to everyone.
  • Click Create Prompt to save, or select an existing prompt to edit.
  • Use saved prompts in chat by selecting them from the prompt menu or typing the command.
Tip: Shared prompts are visible to all users in your workspace. Private prompts are only visible to you.

Tools

What it does

Tools are functions the AI can call during conversations to perform actions like searching the web, running code, or accessing external services. They come from MCP servers configured by your administrator.

How to use it

  • Toggle individual tools on or off using the checkboxes.
  • Use the server filter dropdown to view tools from a specific MCP server.
  • Use Enable All / Disable All to bulk-toggle tools for a filtered server.
  • Disabled tools will not be available to the AI during your conversations.
Tip: Tool preferences are saved in your browser and apply immediately to new conversations. If you're not sure what a tool does, check its description on the tools page.

Documents

What it does

Documents provide RAG (Retrieval-Augmented Generation) context for the AI. Uploaded files are split into chunks and stored in a vector database so the AI can search and reference them during conversations.

Supported formats

  • PDF — Portable Document Format files.
  • Word — .doc and .docx files.
  • Text — Plain .txt and .md files.
  • YouTube — Import transcripts by pasting a YouTube URL.

How to use it

  • Drag and drop files onto the upload area, or click to browse.
  • Paste a YouTube URL and click Import to add a video transcript.
  • Uploaded files are automatically chunked and indexed for search.
  • The AI will reference your documents when answering questions related to their content.
Tip: Files can be up to 100MB. Larger documents produce more chunks, which improves search coverage. Delete documents you no longer need to keep your workspace tidy.
